On Day 1, I learned the fundamentals of SQL and executed basic queries to retrieve data from tables using Databricks.
🎯 Objectives
	•	Understand what SQL is
	•	Learn how to retrieve data using SELECT
	•	Practice basic filtering
🧠 Concepts Covered
	•	SQL (Structured Query Language) basics
	•	SELECT statement
	•	Selecting specific columns
	•	WHERE clause (basic filtering)
   Practice Queries
   -- Fetch all records
SELECT * FROM employees;

-- Fetch specific columns
SELECT name, salary 
FROM employees;

-- Apply filtering
SELECT name, salary 
FROM employees 
WHERE salary > 50000;
